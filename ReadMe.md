# Asset Tracker

A simple asset management system to track and manage organizational assets.

## About

Asset Tracker helps you keep a record of all your assets in one place. You can add, update, delete, and search for assets easily.

## Features

- Add new assets with details (name, type, location, status)
- View list of all assets
- Update asset information
- Delete assets
- Search and filter assets
